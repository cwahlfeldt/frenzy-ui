Navigator Component

The fz-navigator component provides a responsive navigation system with hamburger menu, search functionality, and customizable content.

Menu Structure

The navigator now supports three menu sections:

  • Eyebrow Menu - Small utility links at the very top
  • Primary Menu - Main navigation shown horizontally on desktop
  • Secondary Menu - Additional menu items in the side drawer

Implementation Options

You can define menus in multiple ways:

  • Using the menu-data attribute with a JSON string
  • Using the setMenuData() JavaScript method
  • Using traditional HTML with slots (still supported)

Position Types

The navigator now supports two position types:

  • Fixed (default) - Navigator is fixed to the top of the viewport
  • Static - Navigator is part of the document flow, scrolling with content

Set the position type with the position-type attribute: 

<fz-navigator position-type="static">...</fz-navigator>
