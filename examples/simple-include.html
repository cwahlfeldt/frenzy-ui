Frenzy UI - Simple Integration Example

This example demonstrates how to use Frenzy UI components with a single script tag.

Integration Code

All you need is a single script tag to include the entire library:

<script src="path/to/frenzy-ui/bundle.js"></script>

Advanced Usage

For more control, you can listen for the initialization event:

// Listen for when Frenzy UI is fully loaded
window.addEventListener('frenzy-ui-ready', () => {
  console.log('Frenzy UI is ready!');
  // Initialize your app here
});
