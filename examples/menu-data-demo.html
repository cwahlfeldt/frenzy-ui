Using JSON Menu Data

                
You can define your menu structure in three ways:

                
    
                    
  1. As a JSON string in the menu-data attribute
    2. 
                    
  2. By using the setMenuData() method with a JavaScript object
    3. 
                    
  3. By using the setMenuData() method with a JSON string
    4. 
                


                
Example Code

                
// Get navigator instance
const navigator = document.querySelector('fz-navigator');

// Set menu data using JavaScript object
navigator.setMenuData({
  eyebrow: [
    { label: "Contact", url: "#contact" },
    { label: "Support", url: "#support" }
  ],
  primary: [
    { label: "Home", url: "#" },
    { label: "Products", url: "#products" }
  ],
  secondary: [
    { label: "About", url: "#about" },
    { label: "Blog", url: "#blog" }
  ]
});

// Or set using attribute (JSON string)
navigator.setAttribute('menu-data', JSON.stringify(menuData));
                


                
Menu Structure

                
The menu data structure consists of three main sections:

                
    
                    
  • eyebrow - Top utility links (optional)
    • 
                    
  • primary - Main horizontal navigation items (optional)
    • 
                    
  • secondary - Additional items for the drawer menu (optional)
    • 
                

                
Each section contains an array of menu items. Menu items can have children for dropdown functionality.
