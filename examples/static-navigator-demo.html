Static Navigator Demo

This demo showcases the Frenzy UI Navigator component with the new static positioning option, allowing it to be embedded within page content rather than fixed at the top.

Static Navigator

The navigator below uses position-type="static" to embed it within the page flow instead of fixing it to the top of the viewport.

Unlike a fixed navigator, this one remains part of the document flow, allowing content to scroll past it. It's perfect for sections where you want a navigation bar that stays with a specific content area rather than the whole page.

Features

The static navigator supports all the same features as the fixed navigator:

  • Menu toggle for mobile
  • Search functionality
  • Customizable desktop navigation
  • Eyebrow menu for utility links
  • JSON or HTML-based configuration

The only difference is that it doesn't affix itself to the viewport, allowing it to scroll with the page content.

Usage

To use the static navigator, simply add the position-type="static" attribute: 

<fz-navigator position-type="static">
  <!-- Navigator content here -->
</fz-navigator>

This makes the navigator behave like a normal block element in the document flow, with the menu dropdown positioned absolutely relative to it.

Example Use Cases

  • Section-specific navigation areas
  • Content dashboards with their own navigation
  • Multi-panel interfaces where each panel needs its own controls
  • Wizard-style flows where navigation controls should stay with a specific step
  • Embedded web applications with their own navigation context
