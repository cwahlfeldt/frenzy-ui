Enhanced Navigator Component

This demo showcases the improved FZ Navigator component with desktop navigation items that collapse into the mobile menu on smaller screens.

New Features

Horizontal Desktop Navigation

The enhanced navigator now displays a horizontal navigation menu on desktop screens with your most important links.

Responsive Behavior

Desktop navigation items automatically collapse into the mobile drawer menu on smaller screens.

Nested Menus

Support for dropdown menus in the drawer with accordion-style animations.

Testing Tips

Resize Your Browser

Try resizing your browser window to see how the navigation adapts to different screen sizes. On smaller screens, the horizontal navigation disappears and all items are accessible via the hamburger menu.

Mobile Navigation

Click the hamburger menu icon to open the drawer menu. On mobile, you'll see the main navigation items at the top of the drawer, followed by any additional menu items.

Implementation

HTML Structure

The enhanced navigator uses the following slot structure:

<fz-navigator>
  <!-- Logo slot -->
  <a href="#" slot="logo">...</a>
  
  <!-- Desktop navigation items slot -->
  <ul slot="nav-items" class="nav-items">
    <li><a href="#">Item 1</a></li>
    <li><a href="#">Item 2</a></li>
  </ul>
  
  <!-- Drawer menu content slot -->
  <div slot="menu-content">
    <!-- Additional menu items here -->
  </div>
</fz-navigator>
